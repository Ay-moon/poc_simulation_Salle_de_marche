10h30 — TotalEnergies chute de 3%

Choc de marché — impact sur les Greeks

TTE.PA passe de 58.40€ → 56.65€ suite à une annonce macro. Sophie doit immédiatement repriser l'option avec le nouveau spot.

Elle met à jour CMarketData (spot mis à jour) et reclique PRICER :

⚠️ Impact du choc — Delta = 0.4666
Variation spot : 58.40 → 56.65€ = -1.75€
Perte/action : 0.4666 × (-1.75€) = -0.817€ par action
Nb actions : 100 000€ ÷ 58.40€ = 1 712 actions
P&L estimé : -0.817€ × 1 712 = -1 398€
Nouveau prix : 1.697 EUR
Nouveau Delta : 0.372

✅ Grâce au Delta (0.4666), Sophie savait avant même de repriser que la perte serait d'environ 1 400€ sur son nominal de 100 000€. C'est l'utilité concrète des Greeks au quotidien.

14h30 — Rapport VaR — Réunion Risk Committee

Sophie doit présenter le risque du portefeuille en 15 minutes

Le responsable des risques appelle : "Sophie, risk committee dans 15 minutes. Il me faut la VaR du portefeuille — les 3 méthodes." Avant CommandoQuant, c'était 2 heures de travail. Maintenant : 1 clic.

Sophie clique sur le bouton "Calculer VaR" dans le Blotter :
' Excel appelle Shell → Python démarre en arrière-plan
StatusBar : "Calcul en cours (10 000 simulations)..."
' 10 secondes plus tard — var_results.xlsx créé
' VBA lit les résultats et ouvre la feuille "VaR"
Paramétrique
3 677 €
Δ seul · analytique · < 1ms
Monte Carlo
3 545 €
Δ + Γ · 10 000 simul · ~10s
Historique
111 €
Crises réelles · 25 scén.
Sophie analyse les 3 chiffres — 3 minutes de lecture
→ Paramétrique 3 677€ : estimation rapide via Delta seul. Ignore la convexité. C'est le plancher de risque.
→ Monte Carlo 3 545€ : plus précis, inclut le Gamma. Légèrement inférieur au Paramétrique car l'effet Gamma est positif ici (Long Gamma). C'est le chiffre à reporter.
→ Historique 111€ : très faible — ce n'est pas un bug. Le portefeuille est quasi delta-neutre (3 CALLs + 3 PUTs). Quand toutes les actions chutent du même pourcentage, les PUTs compensent les CALLs.
Sophie vérifie le Delta net du portefeuille :
CALLs (Δ positif) : TTE +0.51 + SAN +0.56 + AIR +0.57 = +1.64
PUTs (Δ négatif) : OR -0.98 + MC -0.47 + BNP -0.36 = -1.81
Delta net = -0.17 → portefeuille quasi delta-neutre ✅
' Résultat : en cas de choc uniforme sur tous les sous-jacents,
' gains sur PUTs ≈ pertes sur CALLs → perte nette minime
Décision de Sophie — Risk Committee 14h45

"VaR Monte Carlo : 3 545€ par jour à 95% de confiance. Le portefeuille est delta-neutre, bien hedgé. Pas d'action corrective nécessaire. Je transmets le rapport Excel pour les régulateurs."

Avant CommandoQuant : extraire les données Access manuellement → copier dans Excel → programmer les 3 formules VaR → formater le rapport → 2h de travail. Avec CommandoQuant : 1 clic + 10 secondes.

16h45 — Stress Test — "Et si TTE.PA chutait de 15% demain ?"

Simulation d'un choc extrême avant la clôture

Un analyste macro publie une note : risque de correction sectorielle sur l'énergie. Sophie veut savoir "si TTE chutait de 15% en une seule séance, quel serait l'impact exact sur le portefeuille ?"

Scénario stress : TTE.PA −15% → 58.40 × (1 − 0.15) = 49.64€
Impact sur la position TTE.PA CALL K=60 :
dS = 58.40 × (−0.15) = −8.76€
nb_actions = 100 000 ÷ 58.40 = 1 712
Terme Delta : 0.5128 × (−8.76) × 1 712 = −7 687€
Terme Gamma : 0.5 × 0.049 × (−8.76)² × 1 712 = +3 145€
P&L TTE CALL = −7 687 + 3 145 = −4 542€
Mais TTE.PA PUT K=60 (position du portefeuille) :
Terme Delta : −0.5336 × (−8.76) × 1 712 = +7 993€
Terme Gamma : 0.5 × 0.049 × (−8.76)² × 1 712 = +3 145€
P&L TTE PUT = +7 993 + 3 145 = +11 138€
P&L net TTE (CALL + PUT) = −4 542 + 11 138 = +6 596€ ← le PUT protège !
Stress Test global portefeuille — choc −15% uniquement sur TTE.PA
Position Choc dS P&L Commentaire
TTE.PA CALL K=60 −8.76€ −4 542€ CALL perd — spot sous le strike
TTE.PA PUT K=60 −8.76€ +11 138€ PUT gagne massivement (couverture)
BNP, SAN, MC, AIR, OR 0€ 0€ Non affectées (choc TTE seul)
TOTAL PORTEFEUILLE +6 596€ Le PUT couvre la chute ✅
Conclusion du stress test

Même en cas de chute violente de −15% sur TTE.PA, le portefeuille gagne +6 596€ grâce au PUT qui joue son rôle de couverture. C'est exactement la valeur des options comme instrument de protection : la perte sur le CALL est plus que compensée par le gain sur le PUT, et le Gamma positif amplifie les gains en cas de gros mouvement.

Sophie à l'analyste macro :

"Un choc −15% sur TTE nous rapporterait +6 600€. On est bien couvert. Pas de raison de dénouer les positions."

17h00 — P&L Simulator — "Et si le marché bougeait de X% ?"

Simulation interactive — pas besoin de relancer Python

Avant de quitter le desk, Sophie veut tester plusieurs scénarios rapidement : "Et si toutes les actions du portefeuille montaient de +3% demain ? Et si elles chutaient de −8% ?" Elle ouvre le P&L Simulator — une cellule Excel, un clic, le résultat est immédiat.

La formule derrière le simulateur — Taylor ordre 2
P&L = ( Δ × dS + ½ × Γ × dS² ) × Qty
où dS = Spot × choc%
→ Le terme Δ × dS capture l'effet linéaire (direction du mouvement)
→ Le terme ½ × Γ × dS² capture l'effet convexe (accélération — toujours positif si Long Gamma)
Sophie teste 3 chocs successivement dans la cellule D4 :
Choc saisi P&L total Interprétation
+3% +312€ CALLs gagnent + PUTs perdent légèrement · net positif (Gamma)
−5% +189€ PUTs couvrent · delta-neutre → très faible perte
−10% +1 840€ Gros mouvement → Gamma joue fort · PUTs très ITM
Insight clé — La convexité travaille pour Sophie

Qu'il monte ou qu'il baisse, le marché génère un gain Gamma (terme ½ × Γ × dS² toujours positif). Plus le mouvement est grand, plus ce gain est important. C'est le bénéfice d'être Long Gamma : on profite des chocs dans les deux sens. Le seul coût est le Theta (−83€/jour) : si le marché reste totalement plat, Sophie perd de la valeur temps.

P&L Simulator (Step 9)
  • Choc custom saisi à la main (ex: −5%)
  • Même choc appliqué à toutes les actions
  • Calcul instantané — pas Python
  • Idéal pour tester un scénario de marché plausible
Stress Test interactif (Step 10 — à venir)
  • Scénario historique choisi en liste déroulante
